1. Контейнер страницы имеет ширину 960рх, больше 1024рхю
2. Для проверки ширины используется media queries, если ширина меньше 960рх, в этом случае макет становится резиновым, вместо фиксированной ширины.
3. Если ширина браузера меньше 650 зч, то контейнер с классом class="content" растягивается на всю ширину окна, а right SideBar c class="headings" смещается под контейнер class="content".
4. При размере экрана меньше 960 рх (резиновый макет).
Правила для резинового макета:
- wrapepr: 95%;
- content: 60%;
- headings: 30%.
